## 도커 설치

## 도커 엔진 시작

- `$ sudo systemctl start docker`

## 도커 엔진 종료

- `$ sudo systemctl stop docker`

## 자동 실행 설정

- `$ sudo systemctl enable docker`

## Docker의 기본 명령어

```zsh
$ docker [커맨드] [대상]
$ docker [상위커맨드] [하위커맨드] [대상]
$ docker [상위커맨드] [하위커맨드] [옵션] [대상] [인자]
```

## 도커 생명 주기

```zsh
$ docker run --name apa000ex1 -d httpd
$ docker ps
$ docker stop apa000ex1
$ docker ps
$ docker ps -a
$ docker rm apa000ex1
```
